Progressive Autonomy — Inbox Wireframes

The agent starts as a draft assistant and evolves into an autonomous operator. The UI transforms as the agent grows: list → cards → conversation. Four stages over ~8 weeks.

Stage 1 Week 0 — "I'm learning"

Agent handles 10% · User reviews everything · Full list view

What's happening: Creator just connected Instagram. Agent sorted messages and created drafts for all non-sensitive conversations. Every draft needs review. Agent is learning tone, priorities, and business context from every edit the user makes. The front is dense — every message visible with inline drafts.
